MallManagement - Desktop App Quản lý Trung tâm Thương mại

1 Cài MySQL (nếu chưa có)
2 Mở MySQL Workbench
3 Tạo schema mới:  malldb
4 Mở File -> Run SQL Script -> chọn file database/malldb.sql -> Execute
5 Mở project trong NetBeans (hoặc IntelliJ)
6 Đảm bảo file mysql-connector.jar nằm trong thư mục lib/
7️ Mở App.java và Run Project

Nhớ add mấy file jar vào libraries
